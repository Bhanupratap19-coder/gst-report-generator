# GST Report Generator 🧾

Automated GST Monthly Report Generator for CA Firms using Python & Pandas.

## Features
- Auto cleans missing data
- Paid vs Pending clients filter
- Month wise GST summary
- Professional Excel report with colors
- 3 sheets — Full Data, Month Report, Pending Clients

## Technologies Used
- Python
- Pandas
- OpenPyXL
